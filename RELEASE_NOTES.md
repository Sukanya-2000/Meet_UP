# CyberNest 1.0.0

CyberNest 1.0.0 is the first production release across web, Android, API, workers, and administration.

Appearance preferences are available under Settings → Appearance and apply immediately. Signed-in preferences synchronize with the API while local device storage keeps the experience available offline.

Android artifacts use version name `1.0.0` and build number `1`. iOS uses the same release metadata and remains ready for signing in an Apple-authorized environment.

Before public deployment, operators must supply production secrets and provider credentials, configure DNS/TLS and stores, run database migrations and backups, and complete store signing and review.
